WeatherTop About Page

WeatherTop is a web application that will allow the user to add a weather Station to it and then record the readings for that station.

Summary cards will be provided for each station showing the latest readings and trending information over the last three readings that are recorded.

Information that can be recorded is the general weather conditions, temperature, wind speed, wind direction and atmospheric pressure.
